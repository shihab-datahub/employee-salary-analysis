# Employee Salary Analysis

## Project Overview

This project analyzes employee salary data across different departments and cities using SQL. The goal is to understand salary distribution, workforce distribution, and experience levels within the organization.

## Dataset

The dataset contains employee information including:

- employee_id
- department
- age
- salary
- experience
- city

## Business Questions

1. What is the average salary for each department?
2. Which department has the highest salary levels?
3. Which city has the most employees?
4. What is the average experience across departments?

## SQL Techniques Used

- AVG()
- COUNT()
- MAX()
- GROUP BY
- Aggregation

## Key Insights

- The Finance department has the highest salary levels in the dataset.
- IT employees show competitive salaries with moderate experience levels.
- Bangalore and Mumbai appear as major employee locations.
- Marketing department employees have comparatively lower salary levels.

## Skills Demonstrated

- SQL Querying
- Data Aggregation
- Business Data Analysis
- Insight Generation
